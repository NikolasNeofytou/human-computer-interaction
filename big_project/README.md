## âš ï¸ Important Reminders

1. **Keep it simple** - Focus on doing a few things well
2. **Avoid saturated categories** - No fitness, nutrition, heavy booking/shopping
3. **Leverage mobile capabilities** - GPS, camera, sensors, gestures
4. **Think interaction-first** - How is the UX better/different?
5. **Minimal backend** - Focus on front-end interaction design
6. **Low-fidelity only** - Don't create full Figma prototype yet
7. **No grade impact** - This phase is for feedback only
8. **Can change ideas** - But won't get feedback on new idea
