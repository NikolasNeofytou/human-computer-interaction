## ğŸ’¡ PRO TIPS FOR SPEED

### Figma Shortcuts
- `F` = Frame tool (create iPhone frames fast)
- `R` = Rectangle
- `T` = Text
- `O` = Ellipse/Circle
- `L` = Line
- `Cmd/Ctrl + D` = Duplicate
- `Alt + Drag` = Duplicate while dragging
- `Shift + S` = Sticky note (annotations)
- `Alt + 2` = Assets panel (library)

### Time Savers
- **Import Material Design 3 kit** from Figma Community (saves 1-2 hours on atoms)
- **Use plugins**: Iconify (icons), Content Reel (fake data), Stark (contrast check)
- **Copy/paste between frames** instead of rebuilding (Cmd+C, Cmd+V)
- **Create 1 card, duplicate 6 times** for similar screens
- **Smart Animate everything** (saves manual animation notes)

### Quality Savers
- **8px grid** (View â†’ Layout Grids) - auto-align everything
- **Color/Text styles first** - prevents inconsistency later
- **Component variants** - don't create separate components
- **Test prototype after every 2 flows** - catch breaks early
