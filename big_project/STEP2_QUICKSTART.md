### Screen 3: Camera Gestures (3 annotations)

**Annotation 1** (Blue sticky note):
```
Input Prompt Pattern

Clear visual cue shows hand position target with real-time feedback overlay. User sees dotted circle indicating where to place hand, with skeleton overlay confirming detection.

Reference: Tidwell, J. (2020). Designing Interfaces (3rd ed.), Chapter 7: Input Patterns
```

**Annotation 2** (Green sticky note):
```
Progress Indicator Pattern

Recognition confidence meter (0-100%) shows gesture detection progress. Circular progress bar provides continuous feedback until threshold (80%) is reached.

Reference: Tidwell, J. (2020). Designing Interfaces (3rd ed.), Chapter 8: Showing Progress
```

**Annotation 3** (Orange sticky note):
```
Status Feedback Pattern

Success/fail states with haptic + visual + audio confirmation. Text updates from 'Wave your hand...' to 'Recognized!' with green checkmark animation and haptic pulse.

Reference: Tidwell, J. (2020). Designing Interfaces (3rd ed.), Chapter 9: Providing Feedback
```

### Screen 5: Victory Photo (3 annotations)

**Annotation 4** (Blue sticky note):
```
Celebration Pattern

Confetti animation + trophy modal + photo frame creates memorable victory moment. Multi-sensory feedback (visual confetti + haptic burst + audio fanfare) reinforces achievement.

Reference: Tidwell, J. (2020). Designing Interfaces (3rd ed.), Chapter 10: Making It Feel Right
```

**Annotation 5** (Green sticky note):
```
Countdown Pattern

3-2-1 timer with camera preview gives user preparation time. Large numbers with countdown sound and haptic ticks prepare user for photo capture moment.

Reference: Tidwell, J. (2020). Designing Interfaces (3rd ed.), Chapter 8: Showing Progress
```

**Annotation 6** (Orange sticky note):
```
Social Sharing Pattern

One-tap share to social platforms with AR effects embedded. After capture, share sheet presents Instagram, WhatsApp, Messages with photo pre-attached and caption pre-filled.

Reference: Tidwell, J. (2020). Designing Interfaces (3rd ed.), Chapter 11: Social Patterns
```

### Screen 7: Adaptive Dashboard (3 annotations)

**Annotation 7** (Blue sticky note):
```
Dashboard Pattern

Real-time stats (win rate, gesture accuracy, play time) in glanceable cards. Key metrics presented with data visualizations for quick scanning and trend identification.

Reference: Tidwell, J. (2020). Designing Interfaces (3rd ed.), Chapter 12: Information Graphics
```

**Annotation 8** (Green sticky note):
```
Data Visualization Pattern

Line graph shows skill progression over 7-day window. Trend line with data points allows users to track performance improvement and identify patterns (e.g., better on weekends).

Reference: Tidwell, J. (2020). Designing Interfaces (3rd ed.), Chapter 12: Information Graphics
```

**Annotation 9** (Orange sticky note):
```
Progressive Disclosure Pattern

Expand/collapse gesture vocabulary with performance details. Tap 'Gesture Accuracy' card to drill down into individual gesture success rates (shake: 85%, swipe: 72%, etc.).

Reference: Tidwell, J. (2020). Designing Interfaces (3rd ed.), Chapter 6: Getting Around
```

## üí° ANNOTATION TIPS

1. **Placement**: Put sticky notes NEAR the UI element they describe (not on top)
2. **Color coding**: Use consistent colors (blue/green/orange) for visual organization
3. **Size**: Make sticky notes large enough to read (200-300px wide)
4. **Spacing**: Don't overlap annotations ‚Äî leave clear reading space
5. **Arrow tool** (`Shift + L`): Draw arrows from sticky notes to specific UI elements (optional but helpful)
